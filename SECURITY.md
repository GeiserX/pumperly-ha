# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly by emailing **support@pumperly.com**.

Do **not** open a public issue for security vulnerabilities.

You should receive a response within 48 hours. We will work with you to understand the scope of the issue and develop a fix before any public disclosure.
